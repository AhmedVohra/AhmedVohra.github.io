How to use regex in Business Central

To use regex in Business Central follow these steps:

  1. Import codeunit 'Regex': Regex : Codeunit Regex
  2. Define Pattern: Pattern, Value : Text
    Pattern := '^0+'
    * this pattern removes leading 0's from a string
  3. Use Regex.Replace to replace the old string with the new: Regex.Replace('Old String',Pattern,'New String')
    If you want to remove the leading 0's: Regex.Replace('Old String',Pattern,'')
